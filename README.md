## Ethereum ERC20 Contract Demo

This repo contains a ERC20 smartcontract to create Fixed Supply Tokens.
* Owner controls onboarding of participants.
* Token Supply is fixed at the time of contract deployment - no mint operations.
* Tokens created cannot to destroyed - no burn operations.
* Particapants have access to all standard ERC20 interface operations.
* Approval limits can be increased for a spender.
* Struct has been implemented to keep track of all the participants and their balances.
* [Remix](https://remix.ethereum.org) and Metamask can be used to deploy on desired Testnet/Mainnet.
